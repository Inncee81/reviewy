A ultra high HD IP camrea with waterproof design from Amcrest

 I was looking for a replacement for my old IP camera because of its unreliable performance so I purchased this Amcrest ProHD Outdoor 3-Megapixel WiFi Wireless IP Security Bullet Camera after doing my research on Amazon. I’m looking for an IP camera that can provide consistent monitoring with HD quality and this camera does meet my needs.The package includes Amcrest IP3M-943W, power adapter, WIFI antenna, IP camera wall mounting kit, Ethernet cable, and user manual. Amcrest IP3M-943W is waterproof rated at IP67, which means that it is fully protected from dust (6) and can also withstand being submerged in 1m (about 3.3 feet) of static water for up to 30 mins (7) so it should last throw rains without a problem. There are many IR LEDs to help night vision and it’s claimed to do up to 98 feet at night. I don’t need to check that far but I am able to see 30 feet away from my front door clearly at night. The mounting kit includes both anchors and screws and Amcrest youtube channel clearly shows how to install the unit. I like that it come with a long and high quality WIFI antenna to ensure stable network connection. Be note that this unit doesn’t have pan/tile features and angle adjustment is made manually on the unit itself. There was no memory card included in my package so I had to supply my own.Installation:Hardware wise, I just had to open up the memory card slot by unscrew 2 screws. After inserting memory card and finishing first-time setup, I just had to mount the unit and point it to where I want to capture. Connect it to power using the included power adapter and it’s ready to go. Do note that there are 2 ways to mount the unit, one is to hide the wire inside the wall (wiring container is included) and the other is leave the wire exposed.Software wise, Amcrest does a good job helping user to setup. The quick start guide is clear and I followed the instruction to install “Amcrest View Pro” app onto my smartphone/tablet for an easier setup. By scanning QR code on the IP camera itself, the app could allocate the camera through serial number and connect to it to finalize network setup (Pick the desired wireless AP and password). After that, the app can quickly access the camera’s live view. I’d recommend using wired Ethernet cable setup method to setup to avoid complications. Please see my video for a short walk-through.PROs:- High quality video with minimal delay. When viewing the live view in SD quality, the video has almost no delay and it has less than 1 second delay when in HD mode. Please note latency also depends on the quality of the network this IP camera connected to.- User friendly setup and fast customer support. I was trying to setup using wireless method first time and couldn’t get it to work because my home network configuration is very complicated. I contacted Amcrest support and they helped me go through it easily using the wired way. I also found there are a lot of tutorials on Youtube regarding Amcrest IP camera.- Amcrest cloud support and free dyndns. Amcrest IP camera comes with free 4 hour cloud recording and free dynamic dns service. Do note that the free cloud recording is limited to 1 camera per account unless you upgrade to paid membership. Dynamic dns is a very handy service if you’re trying to access the camera from outside home network without a static IP.- Third party IP camera app support. I use Tinycam monitor as my main IP cam app and I found Amcrest camera is supported in the default device list. I forwarded the HTTP port and RTSP port and it’s ready in my Tinycam monitor app. By default it also has iOS/Android app support and live view/setup can be done through browser on PC as well.- Built-in notification alert if you choose to use Amcrest View Pro app. I setup email alert with motion detection because I prefer this way.- Ultra HD resolution. It supports 2304×1296 with H.264 Compression. It’s even higher than 1080P.Note:- Amcrest IP3M-943 only supports 2.4GHz WIFI frequency. No 5GHz support. WEP, WPA, WPA2 Encryptions are supported though.- I would recommend setup the camera first time with memory card slot cover removed because status indicator LEDs are located INSIDE the cover and that’s the only way to see if the unit is running/ connecting/ connected/ off.Overall it’s a very solid IP camera and I would recommend this to anyone who is looking for ultra HD video quality and has no need for pan/tilt features.
