Great kit but the poor quality handle makes this kit useless

This had the appearance of a really nice kit with all the key tools you’d need for a traxxas model in one handy travel container, but we quickly realized the handle was a complete failure which makes nearly all the tools useless. The handle worked well initially easily allowing a number of tool changes, but after a few minutes the tool retention collar on the handle quickly slowed progress and couldn’t make it through a single repair. In the end, I wound up struggling more with the tool kit than my RC car making the tinkering very unpleasant. It was really disappointing. I wouldn’t recommend this kit until they can upgrade the handle a bit.
