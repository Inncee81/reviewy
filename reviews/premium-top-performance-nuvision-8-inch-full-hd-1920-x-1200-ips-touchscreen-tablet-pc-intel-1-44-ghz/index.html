Not the fastest, but it’s amazing for the price!

You cannot beat this device’s value! If you want a windows 10 tablet but you don’t want to spend your entire tax return, look no further.I use this tablet for a program called VCDS which is a diagnostics software for VW-Audi group cars. For me this tablet does a much better job, and is way more convenient, than running the windows-only software through bootcamp on my MacBook.I also have had great experiences emulating old console games using emulators for N64, PS1, and GBA.Newer games (within the last decade or so) probably won’t run as smoothly. If you are looking for a gaming tablet, you don’t have much choice but to buy an expensive tablet.Any program for office work (word processing, programming IDEs, FTP clients, etc.) all work wonderfully. I would recommend this for remote work when you don’t or can’t bring a laptop with you.The battery life isn’t incredible. But I have personally had it last about 5 and a half hours with light use (web browser viewing simple web pages [minimal js and no embeded video] and a word processor open). If you play an emulator or watch YouTube videos, the battery life lasts for about 3 and half to 4 hours.I typically use it until it gets to about 10-15% and then I continue using it with a portable USB power bank (I use an Anker battery). That way it will last you all day no matter what you are doing.Just be sure to remember that there is only one USB port and you can’t use any USB peripherals while charging. I have tested an OTG charging cable that splits the USB I/O and USB charging input for android phones with no luck. So be sure to get a Bluetooth keyboard and mouse (ones that don’t have a USB adapter) to use while you charge.The HD screen is bright and has vivid colors. It blew away my expectations. The audio quality however is very subpar. The built in speaker is downright bad and sounds like a crappy laptop’s speaker with water damage. This may be a deal-breaker for some, but it doesn’t bother me in the least. 90% of the time, I use headphones with the tablet. The other 10%, it is connected to a Bose Mini Soundlink. Even at the pricepoint, it is surprising that the manufacturer couldn’t cheaply source a slightly higher quality speaker. But it is very much a budget tablet.Overall, I would give this device 4.5 stars. It would be perfect if it had a full USB (3.0 or a USB-C would be awesome) and a micro USB for charging. It’s not a huge deal for me, so I only subtract half a star. For the price, it is very well built and does a great job with non-intensive programs, I highly recommend.
