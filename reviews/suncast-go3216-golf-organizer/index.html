Get some help to assemble this, otherwise this is a good product.

You will need help to assemble this unit properly otherwise I would have given this five stars. I assembled this alone and it took me longer than it should have because the fasteners provided and used in the design are flush mounted so as not to have any fasteners protruding from the rack after assembled. Also, one of the casters would not screw in because the mig weld on the nut warped the nut itself on the leg and the threads were screwed up. I know this because I worked in a metals furniture factory for years. The finished unit is sturdy, looks very good and holds all of the golf gear that we put on it with room to spare. Nice Organizer with some minor upgrading.

I shopped around before settling on this organizer. The price was reasonable and it seemed to be an adequate size for my one set of clubs, bag and pull cart. I had read other feedback about it not being too strong so I added some minor reinforcement. I had some 1⁄8 inch thick fiberboard that I added to the bottoms of each shelf. I also added it to the back and sides where the bag and clubs go. I used some #6 machine screws with washers and star washers. I anticipate a long life for this rack as I’m 59 and no one else uses it but me. I don’t throw stuff into it. I put them in. With the protective boards I don’t expect much if any damage.READ MORE

Pretty much as described

Overall, I like it, but there were a few flaws, and it’s not quite as useful as it could be … thus four stars instead of five.I put it in the corner of my garage and it looks nice. I have a cart bag and a few extra clubs, umbrellas, etc., all of which sit quite nicely in the left side.I can get two pairs of shoes in the bottom shelf if I stack them atop each other. Only one pair in the second shelf, but I only have three pairs, so that’s just right. The third shelf is good for golf balls … room for several dozen. The gage of the metal is adequate, but I think it should have been a bit better for something that costs $70+, and there is a crease in the mesh in one area (alhtough it is not noticeable when in use) … otherwise, I found the product quality (fit, finish, etc.) to be good. One of the pieces of foam trim was not installed correctly so I’ll have to get some double-sided tape to reinstall it, but I consider that a very minor issue.I had considered building one out of wood but decided to take the easy way out.
