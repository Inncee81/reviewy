For Android Marshmallow SD internal storage feature, this is the one you’re looking for.

Android Marshmallow introduced a feature that allows you to place entire apps on an SD card, not just storing the data. The minimum specs are pretty forgiving, but I learned the hard way a standard SD-XC wasn’t going to cut it. There was massive lag, if the apps ran at all. This one however feels like it runs almost faster than the native internal storage! If you’re looking to utilize an SD card as Internal Memory in an Android Device, you want this one.
