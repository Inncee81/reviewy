Wireless and it works!

I got this speedometer/odometer to mainly to measure the miles I had ridden. The instructions for installation weren’t very clear and were a little confusing and they are for a wireless as well as wired bike computer. Also, while it tells you to put on the double-stick pad on the bottom of the display piece, it doesn’t tell you to do so with the sensor piece, that which I only realized after I had used the zip ties to attach the sensor to the bike. Another thing I did not understand was the tracks on the magnet for the spokes were parallel, rather then V-shape, so they didn’t really fit, hence I only installed one spoke in one of the tracks. All that aside, the device works fine once you set up the position of the magnet and the sensor. It is great that it is wireless (so there are no dangling wires) and the display can be removed/slid off its console so that it can avoid the harsher elements. The display is clear and easy to read. I measured the distance in miles which I rode against the map and it is mostly correct. There is also a clock feature so it tells time which I find useful and it was nice to see the speed at which I was going while I was riding.
