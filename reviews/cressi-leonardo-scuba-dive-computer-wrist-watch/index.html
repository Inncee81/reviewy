it works great. You can find the config

I could not get this to work in my Windows 7 computer at first. I had to change the config.ini file. The file has a line in it as follows: data_path = “/C/Programmi/Cressi PC Interface/database”. Windows 7 will not normally let the software write to this location and if you change the path to another location, it works great. You can find the config.ini file at C:Program Files(x86)/Cressi PC Interfact/config/If you choose a location that you normally backup then your dives will be safe.
