Very fast, highly compatible, perfect as expanded storage for mobile phones that can still use cards like these!

Excellent card - fast, error free. I use this as expansion memory for my Droid Turbo 2 64GB phone, and have directed Android to use it for photo storage, and have set Google Play Music to use it for music downloads and caching. Works flawlessly. Being able to add expanded memory to the phone has changed the way I use it and some of the apps I use. Thinking now that a 128GB version of this would be great – next on my list when the price is reasonable. 64GB, though, is quite a lot of space. I wouldn’t hesitate to recommend this - seems to be an excellent product all around, and probably one of the fastest on the market as of the beginning of 2016.

I am pleased with the performance of this device

I am pleased with the performance of this device. It has a few allocated gigabytes that are not formatted for use but this can be fixed if one knows how. I have been using this SD card for a little while and while I can’t judge the long term viability of it’s life span (this is hard to guess with any device) it is very good and a quick transferring card. I have reformatted it to NTFS without problems although one would have to be careful about what settings are chosen on a flash memory device with NTFS reformatting. All the same this memory card is serving it’s purpose well and I would buy it again if and when another is needed.
