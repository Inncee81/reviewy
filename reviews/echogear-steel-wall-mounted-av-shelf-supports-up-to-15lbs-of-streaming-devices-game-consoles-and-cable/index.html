There should be either screws that match the screw hole size or the mount holes should protrude to lock the shelf in place. If you keep the items centered it likely wont be an issue however. If you handle items a lot on this shelf i would expect it to loosen.You can plainly see what i used this for. It barely fits however does work. Theres simply no wiggle room.
