Product Specifications

  • 10 x 4.5” Self-Tightening Props
  • One Clockwise (CW) Prop

  • One Counterclockwise (CCW) Prop

    Product Description

The Propeller Set for Solo Quadcopter from 3D Robotics features a pair of spare self-tightening props in black, one clockwise (CW) and one counterclockwise (CCW) model. Please ensure, when replacing your props, that you use the same rotational model as was there before. Improperly installed props may loosen and detach in flight.
