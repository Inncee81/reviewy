Product Specifications

  • Unique “zig-zag” feature allows the child to get ready when the ball is released
  • Release lever can be moved to either side for left or right-handed hitters
  • Trainer adjusts from 27” (68.6 cm) ball release height to 45” (114.3 cm) in 6” (15.2 cm) segments
  • Large stable base with two cup holders

  • Includes four balls and a bat

    Product Description

The Home Run Baseball Trainer by Step2 will be a grand slam with your kiddos! This sports toy is a stand-alone trainer that helps kids practice their baseball skills. The Home Run Baseball Trainer can enhance children’s hand-eye coordination as they swing the bat to score a home run. Mini athletes will learn about cooperative play as they take turns going up to bat.
