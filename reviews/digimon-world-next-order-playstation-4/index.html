Both nostalgic and innovative

Absolutely love this game. I admit, the first one made me want to break my PlayStation years ago. But I love that way you raise your digimon affects how they evolve. It brings a very personal touch to the franchise that not all games have. There are a lot of new features in Next 0rder that make the process almost too easy (a bathroom in the gym, a lab that gives you evo prerequisites, etc). However these features save a lot of time and allow the player to accomplish more and take the game further. My lack of one star is only because DORUmon isn’t currently in the game.

Setup Sucks

I have played almost all the digimon games, and pretty much loved them, then I got this game. The way the evolution system setup is by far the worst setup of all the digimon games I have played. You have little to no control how your digimon digivolve. The battle system has response issues, it doesn’t respond to diagonal directions, so its easy to select the wrong attacks.
