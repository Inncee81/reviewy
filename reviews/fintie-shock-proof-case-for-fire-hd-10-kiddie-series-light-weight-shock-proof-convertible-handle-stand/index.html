Handy for carrying.

I purchased this case for one of my Kindle fire tablets. The main reason I bought it was the handle. I’m really glad that I made this purchase. Not only is the handle convenient for carrying, it does a good job of propping the tablet up for hands-off viewing (horizontally). The case is light weight, easy to keep clean and very reasonably priced. You do need to buy a screen protector (especially if a child is using the tablet), but, I like having the screen protector separate from the case, so it can be changed if necessary. The only complaint I have is that it’s not very stable when standing vertically, but I don’t use it much in that position, so not really an issue. I would recommend this case to others.
