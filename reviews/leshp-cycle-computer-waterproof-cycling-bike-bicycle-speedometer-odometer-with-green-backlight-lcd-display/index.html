Product Reviews

Good so far!

I recently installed this but have not had enough time to do a proper review. This is mainly about the setup process.I got this for its backlight. Often I ride at night and it’s useful to have the backlight.The setup instructions were reasonably easy to understand. The sensor magnet is ceramic rather than neodymium so placement is more critical and the gap needed is small. The battery sensor is a bit larger than it would be with neodymium magnet. Placement of the computer itself is also critical. The optimum reading angle is between about 60 and 90 degrees to the face. At 90 degrees the display drops off quickly. Initially I put the computer roughly parallel to my face and found it often hard to read. Repositioning it to about 75 degrees made it easier to read.Setting all the numbers was about as difficult as other computers I have used. There are many features on this computer and it takes a bit of getting used to. So far it seems to be good.100 mile update - still good!
