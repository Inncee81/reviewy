Product Reviews

An excellent set, both main and alternate models

This is a very interesting airport type fire truck, it just screams BUY IT. The main model has 6 wheel steering and the hose on the front via a knob on top of the vehicle allows you to aim it in nearly 180 degrees. Where it excels as well is the alternate models, 2 police type buggy vehicles that are equally as fun, both aesthetically and functionally. All in all, a set worth buying, I have not regretted it at all.

Lego rescue truck

This is a nice lego set looks great. However it is lacking a little the water canon on the eight wheel vehicle gets boring very quickly. On the good side it is lego and if you dont like what they did you can always change it and make it whatever you want it ot be. So keep on building and learning.

Rescue Truck

The Rescue Truck is a strange looking affair about 13” long or so. I like the engine and the 3 axle steering. I can’t see,however, what is new about the steering system. I have dozensof technic sets and it’s nothing new. The water hose can be adjusted (cool), but it’s always breaking. Although this is agood idea for a set, I’d make it bigger (aviation fire trucksare huge!) and I’d make it more realistic. It’s an all right set.

An Awesome Set!!!

Wow, what can I say about this lego set. It is amazing! It has a new type of steering, which has never before been seen in/on a lego set. It also has eight wheels that are each huge. This set is HUGE, with 600+ pieces. I really enjoyed making this set and playing the the pieces to make other sets from my imagination.

Excellent kit with additional instructions for two racers

This is a fun, challenging kit to build with lots of moving parts and a great engine and steering mechanism. It comes with an additional instruction booklet with detailed plans for two Formula 1 racers.
