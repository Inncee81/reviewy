Cronus Tactical

The new Tippmann Cronus Tactical combines high performance with incredible durability in a milsim body.  The Cronus features our reliable in-line bolt system in a high-impact composite body with soft over molded rubber grips.  Three main mods, including a mock silencer, carry handle, and adjustable stock, help complete the military look.  This marker can easily be further modified by mounting an assortment of accessories on any one of the four picatinny rails. 

3Skull Mega Set

Everything you need to get in the game is included in the 3Skull Mega Set.  Here is what is included a 3Skull 20oz Tank, 3Skull Harness, Ultimate Paintball 140rd Tubes, Gold Cup gun oil, Ultimate Paintball 200rd Hopper, 3Skull Orange Squeegee and a GXG Xvsn Mask,

Very reasonable for the price

I got this starter pack for if any of my friends need to borrow a marker or theirs breaks down mid game (or mine knock on wood). For the price, its not a bad package. The one thing to note is that the bottle is a CO2 bottle. There is a package out there on amazon that has an HPA bottle, you will see that it has an air guage on the regulator and it costs a little more.ProsIts super cheap. It will get you running until you can try out other equipment to see what you really like. The gun looks cool, and there are some cosmetic upgrades that exist out there. So far the gun has been reasonably consistent and all the peripherals that came with it are workable.ConsThere aren’t any real upgrades that I’ve been able to find easily for the gun other than cosmetic ones so far. So it gets to stay basic, and if you want something with more panache, you will need to invest in a new system. In order to get to the internals, you have to unscrew EVERYTHING and split it in half just to maintain it. Easy to lose parts that way. Can’t change out the feed neck aperture. The mask is a little small (I have a large noodle), but is workable otherwise.Summary:Without spending more money, it’ll be hard to find a better startup deal.
