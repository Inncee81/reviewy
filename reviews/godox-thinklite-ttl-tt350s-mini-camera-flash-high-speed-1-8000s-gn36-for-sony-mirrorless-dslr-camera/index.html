pretty good size and power ratio

Its solid, pretty good size and power ratio. The head only tilts back behind the right shoulder but not the left shoulder which limits bounce flash usability, particularly for weddings. Using it to trigger other godox flashes is hit or miss, most of the time the flash won’t trigger other units until the second or third photo in a series. Sometimes it doesn’t trigger at all for a while, sometime it trigger perfectly for a bunch of shots in a row. Just overall very inconsistent remote triggering when used as a master. If the triggering issue is resolved via firmware, I would give this a 5 star because the value would be incomparable.The UI is different from all other Godox products (just like every other Godox product’s controls are different from all the others, heck the Nikon X1T vs Sony X1T has wildly different controls), but this one is actually pretty decent to use once you get used to it. I wish there was a memory function where you can save settings and quickly recall them. For instance, save setting 1 to set the flash to master remote mode with settings for remote flash a and b, and set setting 2 so it disables remote flashes and enables the master flash to fire for quick OCF to on camera bounce flash swapping (would be amazing for weddings!).
