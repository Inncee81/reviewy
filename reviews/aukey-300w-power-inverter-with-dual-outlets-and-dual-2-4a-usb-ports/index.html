Low cost modified sine wave inverter with good features. But won’t do 300W!

Low cost modified sine wave inverter with good features. I purchased this inverter recently and evaluated it in an electronics lab and real world situations. $20 is a good price for a 300W inverter with the given features. Major note is that this a “modified sine wave output” device (as are most inexpensive inverters these days). The output voltage is not a sine wave but rather a pulse sequence of about 5ms of +145V, 3ms of 0V, 5ms of -145V, 3ms of 0V and then repeat. Output voltage measured as low as 108Vrms and as high as 116Vrms on a true RMS multimeter. Oscilloscope waveforms showed some distortion that got worse at higher loads. Modified sine wave is good enough for resistive loads such as heating and incandescent light bulbs. It may be OK for some electronics. I planned to use this to charge my drone batteries but will not with this output. Voiding the warrantee or damaging batteries is not something I will take a chance on. Modified sine wave can also cause some electronics to be less efficient.I tested the inverter with 25W, 75W and 150W incandescent bulbs single and in combinations. 12V input source was a regulated DC power supply with 50A capability. I set the DC output voltage to 12.0V.With no load on the AC output the inverter drew 0.28A or 3.3W.I have the detailed data but will only summarize input/output efficiency for this review:25W –> 81%, 75W –> 84%, 100W –> 86%, 100W –> 87%, 150W –> 87%, 175 –> 83%It takes about 1s after input power is turned on for output to turn on.The inverter did not seem warm at any time. If I do this again I will add a temperature sensor to the inverter.The inverter shut down during the 225 (150+75) test.The inverter would not start with 250 (150+75+25).The inverter would not smoothly run a desk fan.Another note is that I list the rated value of the bulb. The actual power (output power of inverter) was slightly less for each one but efficiency was 100% * output power measured / input power measured.Not being able to run 225W is disappointing but not unexpected.Note that at 225W, the inverter input current is about 20A. That is a lot of current through a cigarette lighter adapter if you try to use it. Fortunately this inverter comes with the ability to direct wire (using a 30A fuse) to a battery.In conclusion, I won’t be using this inverter for what I intended. And, I don’t know what I will use it for. But it didn’t cost much, was fun to evaluate and likely will work OK for most people. So, 4 Stars
