A Powerful (yet inexpensive) home theater box option…

This kit was my first (of two and eventually more) pi 3 kit and I can’t say enough good about it. A quick install of OSMC or openelec, a microsd card and an HDMI cable and you’ve got yourself a mini-HTPC for peanuts. If you have a plex server at home RASPBMC can be used to really bring everything together. Heatsink is a super-easy install and so is the case. My only regret is not going with the clear case on my first order (but I just like seeing the pi board inside). Everything about the order was easy and quick. Friend of mine ordered one after I told him about it and I’ll probably be ordering more in the future to try some new projects (retropie). Would definitely recommend even to the less technically savvy folks- it isn’t rocket science to get yourself up and running given the extensive documentation available. For ~$50 you are getting a world of fun.

My New Favorite Device

This thing is a new favorite of mine. It’s really an excellent tool, great price, powerful components, great docs from the raspberrypi.org site. This case is well done. Not sure how necessary the heatsinks are in my case, but they shouldn’t hurt anything and were easy to apply with the included instructions.One caution, I ruined one SD card trying to put my board into the case. After getting everything working outside its case, I was trying to shoehorn it into the case with the SD card plugged in, and apparently put enough awkward pressure on it to bend the SD card in half, rendering it dead. But I set up another card the same way (with NOOBS) and was back in business in no time. I don’t really think it’s a design flaw with the case, but the $6 lesson I’ll pass on to others is: unplug everything before installing it into the case.

Great Package

Depending on what you want to use the Pi for, you will most likely need a case, heat sinks, and a power supply. The case would probably earn between 4 and 4.5 stars but is still good quality and not a bad buy on its own. The power supply and heat sinks both do a job and do it well easily earning 5 stars. I did try piecing this together to see if I could save some money but ended up purchasing the whole unit because there was no savings at that time. As a starter kit I would buy this again. Awesome! That is all I have to say about …

Awesome! That is all I have to say about this. The box, heatsink and power supply is of very good quality. I was in the market looking for a small computer where I could install Apache, run as a reverse proxy to run an SSL server to front a couple of video cameras which does not have SSL support. The product comes with a quick start booklet which initially did not work for me. The booklet is based on using a MicroSD card that has OS installed on it. I started from a blank MicroSD card directly from Amazon which caused the unit to not boot on first power up. However, going to the Raspberry PI website and looking at the downloads I could see the options available. From there I was able to find clear instructions on how to download to a MacOS machine and transfer it over to the MicroSD. From there to boot up and installation of Apache proxy server was about 5 minutes. Very impressed. I initially booted the system with a wired LAN but realized that wireless setup was just as easy. I had allocated two days to complete the work but I was pleasantly surprised to be done in 5 minutes. I intend to use the box in a headless mode but during setup I had a taste of the UI running browser and terminal. I can see it becoming a solid general purpose compute unit in no time and I can say bye-bye to my old desktops.
