Great price for Micro SDHC card plus adapter.

Just what I needed for a very reasonable price, and with prompt delivery. When getting a “Micro” SD card, one also needs an adapter so that an SD card reader can read or write data to this new micro format. Be cautious when you see a good price for a Micro card—It may not include the adapter, although you can use the same adapter for different Micro cards. This included both from name-brand SanDisk, and cost less than some Micro card-only (i.e. no adapter) offers. I’ve used this new card a few times and it works great.
