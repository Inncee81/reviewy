Although I haven’t used it yet, but it is good to know it is there when or if I need it.It has a small and compact form factor with a solid albeit plastic build. Again, for the price, I’m not complaining. It has a subtle white LED that doesn’t blind. All the USB ports feel solid. Now, I have read few reviews complaining that using both an external mouse/keyboard and an external HDD causes some lag in mouse/keyboard input, I have done some tests while transfering some files and found no such issues, yet (PS I used the non fast charging port of my laptop). Speaking of tranfering files, I’m happy to report the speeds are consistent and the same using my Transcend 1TB external HDD with USB 3.0.In conclusion, as long as you don’t throw it around like a tennis ball, I’m pretty sure this USB hub will hang around for quite a while.
