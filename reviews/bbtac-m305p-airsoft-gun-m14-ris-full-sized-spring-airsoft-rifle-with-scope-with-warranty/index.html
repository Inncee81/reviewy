    Product Description

Product Description This M305P airsoft gun by BBTac is a exact replica of a M14 rifle. It has a shorter barrel that is great for CQB usage and is similar to the SOCOM version. It comes with RIS and multiple accessories such as a flashlight, scope, and a tactical red dot light with pressure switch. Since this is a spring rifle, it requires you to cock each time to shoot. No battery or gas are required to operate this airsoft gun. Easy to maintain and lots of fun to use for an airsoft war.

If it shoots but no BBs come out, then the problem is either the magazine is not feeding, or something is jammed inside of the barrel. If there is something jammed inside the barrel then you must take out the magazine and slide the unjamming rod inside to get whatever is stuck unstuck. You should be able to see the tip of the rod where the magazine is placed. If you do not see the rod then there is something stuck.
