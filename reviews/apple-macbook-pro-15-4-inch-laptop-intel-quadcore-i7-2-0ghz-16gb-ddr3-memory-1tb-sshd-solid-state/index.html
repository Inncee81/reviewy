I would have liked to return the this laptop and upgrade to a machine that could handle this workload, but the return date was 2 weeks. Do not buy this computer if you plan on using it for heavy duty processes like video gaming, video editing, programming, photoshop or anything GPU heavy. Save your money and get a newer Mac.
