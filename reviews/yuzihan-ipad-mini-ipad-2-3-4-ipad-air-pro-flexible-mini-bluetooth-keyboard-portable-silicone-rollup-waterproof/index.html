Neat Keyboard, but Somewhat Difficult Keys

As I am currently writing this review on this keyboard, it is apparent that it works! The flexibility is really neat, and it’s simple to set up (as long as you don’t confuse the process by looking at the convoluted instructions). It’ll take some time to get used to the small keyboard, but that’s to be expected of any similar device. My only issue with the keyboard is that each keystoke has to be quite deliberate in order to register, which is frustrating when one is used to just fluttering about the keys on a traditional keyboard. I’m going to keep it because I have hope that I’ll be able to get used to the discrepencies and it really is very cool, but I thought other potential buyers should be aware, as the key pressing issue may be significant for some. Haven’t tested its waterproof claim, but it certainly feels as if this could be the case.-Using with Amazon HD Fire 8
