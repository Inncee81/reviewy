    Product Description

This power supply is specifically designed for single board computers such as the Raspberry Pi 3 Model B. It can deliver consistent 2.5A current.

This is compatible with all MicroUSB SBCs including:

  • Raspberry Pi 3 Model B
  • Raspberry Pi 2 Model B
  • Raspberry Pi Model B+
  • Raspberry Pi Model A+
  • Raspberry Pi Zero
  • ODROID-C1+
  • ODROID-C2
  • Banana Pi M3
  • Pine64 It features overtemperature protection and will safely turn off if internal temperatures exceeds a safe threshold. It is capable of safely supplying stable power with up to 3A burst current with voltage drop to 4.8V.

Still get a “lightning bolt” more often than expected

Using my raspberry pi as an owncloud server and started out using a whatever was laying around power supply (cell phone charger-2.0 amp) and the lightning bolt icon came on frequently when system running with small amount of load. When I attached the LoveRPI 2.5a power supply the lightning bolt still appears, but less frequently. The only significant USB power drain through the pi is an SSD. The rest of the system runs on external power. I guess I expected the low-power indicator to completely stop coming on with a 2.5a supply. Either way, the server runs as expected, so I’m not unhappy, but I give 5 stars for “works as expected.”
