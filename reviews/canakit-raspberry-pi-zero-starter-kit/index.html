    Product Description

NOTE: This kit includes the standard version of the Raspberry Pi Zero and does NOT include wireless connectivity.

This exclusive bundle by CanaKit includes the Raspberry Pi Zero and all the essentials! The ultra-small and ultra-slim Raspberry Pi Zero is the smallest form factor Raspberry Pi on the market! It is 40% faster than the original Raspberry Pi but measures only 65mm long by 30mm wide and 5mm deep. The Raspberry Pi Zero support mini connectors to save on space and the 40pin GPIO is unpopulated providing the flexibility to use only the connections your project requires.

Kit Includes: - Raspberry Pi Zero Version 1.3 Board - 8 GB MicroSD Card (Class 10) - Pre-loaded with NOOBS - CanaKit 1A Power Supply - USB OTG Cable - Mini HDMI Adapter

Pi Zero Board Specifications: - 1Ghz, Single-core CPU - 512MB RAM - Mini HDMI and USB On-The-Go ports - Micro USB power - HAT-compatible 40-pin header - Composite video and reset headers - Dimensions: 65mm x 30mm x 5mm

Note: Pi Zero Version 1.3 includes a camera port. The Pi Zero camera port only works with the Raspberry Pi Zero Camera Cable. It does not work with a standard Raspberry Pi Camera Cable.
