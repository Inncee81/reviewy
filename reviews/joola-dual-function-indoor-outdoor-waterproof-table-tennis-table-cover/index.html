    Product Description

The JOOLA Dual Function Indoor/Outdoor Waterproof Table Cover is made to protect your outdoor table from all harsh weather conditions, as well as scratches and dust. The cover is fit with a unique, easy-to-use securing system involving velcro, buckles and straps to ensure 100% protection. Cover your table in both playing and storage position using the velcro to attach the cover and straps to secure to the undercarriage. The JOOLA Dual Function Outdoor Table Cover is made from Polyester with a waterproof PVC coating that creates a plastic-like seal that prevents any precipitation from coming in contact with the table. Use this table cover indoors or outdoors to keep your table tennis table looking good as new and clean for each use!
