Product Reviews

Works good. Kids loved it.

Used for the first last weekend. Had 3 kids using it ages 6-10. It was great for all the ages, especially the little ones. In fact, it is a bit easier to use the smaller the kid. The bigger the kid, the less stable it is. I even tried to get on it and it was tough to balance on it. It would slide out from under me. One thing is to make sure the handle fits the child. If the handle is too long, it pulls back, the nose lifts up and the whole thing flips over backwards. The handle is adjustable so that is good. All in all, a good product to build confidence and work on balancing and manuervering for little ones. Easy to set up and start using.

Loved it

I have 3 boys (7, 5, 3). All of them loved it. Very simple and requires little to no effort. I pulled them at 13 mph behind my Mastercraft x55. This is a great confidence builder, and this is what I was looking for.
