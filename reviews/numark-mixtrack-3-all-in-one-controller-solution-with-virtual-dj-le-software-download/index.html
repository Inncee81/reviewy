    Product Description

style:No RCA Output From Numark—the world’s leading innovator of DJ equipment and technologies—comes the incredible Mixtrack 3 DJ controller, designed with the professionally-aspiring DJ in mind.

This controller doesn’t have a built-in soundcard, so it’s made to connect to whatever external audio interface you plan to combine with it to route sound to speakers and headphones.

Dive into our 16 multi-function backlit performance pads and add loops, launch samples, and control hot cues to your performance. Use the multi-function touch strip for dynamic FX control and track search operation. There are dedicated filter knobs for each channel, to give you just the exact amount of tonal control you need for that perfect sound. If you can think it, Mixtrack 3 will do it.

With the included Virtual DJ LE software you can create professional-sounding mixes at home or easily upgrade to Virtual DJ Pro for performing live at parties, functions and clubs. Plus, Mixtrack 3 operates via standard MIDI to be compatible with virtually any popular DJ software.

  Frequently Asked Questions  

Do I need a driver for the Mixtrack 3? The Mixtrack 3 is class compliant.  It uses device drivers that are already built into all Windows and Mac operating systems.  This means that there are no extra drivers for you to install.  This also means that the Mixtrack 3 will be compatible with the latest Mac and Windows operating systems the minute they come out - no driver updates needed!

 

Which DJ software is included with the Mixtrack 3? The Numark Mixtrack 3 comes bundled with Virtual DJ 8 LE.  This software works seamlessly with the Mixtrack 3 and requires no mapping to use.

The Mixtrack 3 also comes bundled with the Prime Loops remix tool kit for adding loops and samples to live sets and TOOLROOM Artist Packs

To obtain your software downloads, first you will need to register your Mixtrack 3 in your account at Numark.com.  Once you register using your serial number you will have access to the download links and VDJ authorization code.

 

Can the Numark Mixtrack 3 be used with my iPad? The Numark Mixtrack 3 is not officially supported as an iOS peripheral and is not being marketed as an iOS-compatible device.  It is intended to be used with the included Virtual DJ LE software on a supported operating system.  However, as it is a class-compliant MIDI device, it will work with nearly any device that accepts such information.  For example, to use it with an iOS device such as an iPad, an Apple camera connection kit and a powered USB hub would be necessary.

 

What type of pitch sliders are used? The Numark Mixtrack 3 has several pro features.  Class-exclusive 100mm pitch sliders make it simple to execute extremely fine pitch adjustments, so it’s easy to keep two cuts in beat sync without any drift. The conventional 60mm pitch sliders on competitive units can’t come close to matching the MIXTRACK 3’s performance in this critical area.  

A multi-function touch strip for dynamic FX control and Strip Search operation.  The touch strip allows for quick scrubbing of songs to set cues on the fly along with FX modulation.

Independent transport controls for greater operational flexibility, both utilizing metal—not plastic—controls for long-term durability. 

  I lost my software disk or authorization code for the included software.  What should I do? Your Virtual DJ 8 LE code is located in your Numark.com account.  If you need to re-install the Virtiual DJ software, you can log in and get access to the download and authorization code.

Numark does have a process in place where customers can be provided replacement authorization codes and/or download links if they have lost their disk, or encounter difficulty authorizing the included software.

When circumstances allow, replacement authorization codes are issued to customers free of charge.  Basic customer information is required any time a replacement code is requested by an individual. This required information and further contact details can be found on the website below.
