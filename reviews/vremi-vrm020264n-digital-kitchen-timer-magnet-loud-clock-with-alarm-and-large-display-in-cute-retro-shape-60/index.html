Kitchen Timer Details Size: 3.2 x 2.52 x 0.8 inches Includes fridge magnet, support stand and AAA battery Clock Mode Press MODE until Clock icon appears on LCD screen Press HOUR to set Clock hour and MIN to set Clock minutes to current time Press START/STOP to switch b/t 12 or 24hr mode (PM icon appears for 12hr) Countdown Timer Mode Press MODE key until Timer icon appears on LCD screen Press HOUR to set hour, MIN to set minutes and/or SEC to set seconds To start or stop/pause the Timer, press START/STOP Alarm beeps for 1 min if left idle end of Timer - press any key to stop To reset Timer, press/hold MIN and SEC keys at same time until display shows “00:00:00”, then set HOUR, MIN and/or SEC Stopwatch Mode Enter Countdown Timer mode, then press START/STOP for Stopwatch Press the START/STOP to start or pause Stopwatch Alarm beeps for 1 min if left idle at max time “23:59:59” - press any key to stop To reset Stopwatch, press/hold MIN and SEC keys at same time until display shows “00:00:00”.
