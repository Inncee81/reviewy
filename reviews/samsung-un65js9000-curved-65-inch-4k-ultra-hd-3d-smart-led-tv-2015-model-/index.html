You might want to wait for the price to drop, but you won’t be sorry!

I purchased this TV last week locally (Sorry Amazon) after considering the previous 9000 series (UN65HU9000FXZA) for the past couple of months. Is this TV worth $1,500 more than the previous version? (as of this writing in early April, 2015 this model is selling everywhere for $4,999). Probably not, but hey, if you can afford it and you are an early adopter like me, go for it. The picture quality is outstanding! This television is noticeably brighter than the previous model in the head to head,in-store comparisons I did. Some reviewers have complained about reflections on the shiny curved screen. Although this model has the same surface finish, I haven’t noted any major problems with reflections. This TV is plenty bright enough despite being in a family room with lots of big windows and skylights. The colors are extremely vivid and the depth enhancement function is pretty wild as well. As several others have discussed with the previous model, you’ll need to experiment with settings. I’d note that I didn’t like the Movie mode that some reviewers on Amazon recommend.The only complaint I have with it is that the Smart Hub doesn’t include Netflix, which by the way, is about the only source of 4-K programming.at this point. The one connect box has enough (4) HDMI inputs, so I solved that problem with my AppleTV, but it would have been nice for Samsung to build Netflix into the system. Maybe with the next firmware update.Setup was easy, right out of the box. All-in all, you’ll love this TV.

Fantastic, wonderful, glorious TV.

I upgraded my triple monitor setup (1 x 24” and 2 x 23” LCDs) to this display. Fullscreen 4K PC gaming is amazing. The colors are amazingly vivid, the brightness is superb. This was also my first smart TV, and I’m sure those features are relatively standard, but I was still impressed by it. I used to run a projector for my computer gaming (~9.5’ diagonal display) and thought that I was Futureman from that…but damn, this display knocks that out of the park.One downside is that it appears that Samsung relegated the user interfacing drivers to the individual smart TV app makers. What I mean by this is that I could connect a USB keyboard to the breakout/source box (how have we lived so long with TVs without having these things as standard?! Truly revolutionary piece of port donglin’) and the Netflix app picked it up right away. I can use the keyboard to get connected to my account, search for content, do whatever by typing into the keyboard like one would expect. But the Amazon Video app only sees the standard USB keyboard as a set of arrow keys and an enter button. It refuses to see the alphanumeric keys as what they are. Very frustrating to have such a different experience between apps when the hardware layer should be handled by the operating system itself. One could say that this isn’t as much a slight against Samsung and more against the Amazon Video app team, but honestly, basic drivers like this should be uniform and the app makers should just have to hook into the API. Layers of abstraction people…basic concepts.If you’re considering a huge display like this, make sure you get yourself a great wall-mount kit as well. I got this one ECHOGEAR Full Motion Articulating TV Wall Mount Bracket for most 37-70 inch LED, LCD, OLED and Plasma Flat Screen TVs w/ VESA patterns up to 600 x 400 - 16” Extension - EGLF1-BK. Works very well. With two people had it installed in less than an hour. All of the pieces you need are in the kit and comes with the TV.

The set is amazing and the picture is simply beautiful. I’m still fine tuning some of the settings until I’m completely satisfied with the picture. It’s fun to play with the remote, once you learn how to operate it. It’s incredibly fast and accurate. The sound is very good for a television, I still prefer my home theater system for movies and sports. There are so many features on this model that it will take a few more months to learn and enjoy all it has to offer.
