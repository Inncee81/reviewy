Product Description

The new 3rd-generation LEGO EV3 Intelligent Brick is powered by the ARM-9 processor which is significantly more powerful than the previous generation NXT Intelligent Brick. It runs on the Linux OS, an open-source software which give advanced users a highly flexible system to hack or modify.

The EV3 software which is not included can be downloaded freely from LEGO.com is a friendly, graphical design programming language based on National Instruments’ LabView.

Apps are also available for the Apple IOS and Android OS smart phones and tablets for greater control and possibilities with the LEGO EV3.

The LEGO Mindstorms EV3 Intelligent Brick is a wonderful and powerful tool for beginners learning how to program to advanced user creating rather sophisticated robotic systems - it is limited only by your imagination.

The EV3 Brick uses 6 AA Batteries or the EV3 Rechargeable Battery, and both are sold separately.

Compatible with older LEGO NXT sensors.

Lego Mindstorms is a world-wide phenomenon. The heart of the system is the Intelligent Brick, a sophisticated computer that includes analog-to-digital converters for robotic sensors and outputs for motor controllers. It also has an LED screen, input keys, and built-in Bluetooth and Wi-Fi. The most modern version, the EV3, can even interact with other EV3s to enable cooperative robotics.I already had an EV2 system, so I decided to upgrade starting with the EV3 brick itself. It’s a wise choice. My EV2 sensors and motors work perfectly with the new brick, and the Mindstorms programming system offers the same block-based interface used in previous versions. Note that you don’t have to buy the brick separately if you already have an EV3 kit.Mindstorms is a learning experience that can help both adults and kids learn about mechanical design, programming, planning, collaboration, and robotics.
