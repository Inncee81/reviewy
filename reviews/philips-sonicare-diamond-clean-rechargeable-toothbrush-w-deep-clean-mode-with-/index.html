- Cleans significantly better than a manual toothbrush - Charge lasts 3 weeks …

Pros:- Cleans significantly better than a manual toothbrush- Charge lasts 3 weeks- Pulses to let you know when to move on to each section- Travel case is a big win. With my prior electric toothbrushes, I often ended up traveling with a manual just because I didn’t have a convenient way to store the electric- Variety of modes is great for a whole spectrum of dental preferencesCons:- Like others have said, It’s odd that you can’t set the mode prior to turning it on and putting it in your mouth. It leads you to have to either count the number of times you push the button, or angle it so you can see the display in the mirror.- The instructions are very vague regarding the timer and quadrant pulses. I started with using the gum mode and thought I was following along switching quadrants every time it pulsed, but then found myself with an extra 4 or so pulses at the end that I didn’t know what to do with. It would be nice if the instructions had a table that called out specifically when the pulses occur for each mode so we know when to move to the next section.- There’s another model of sonicare that pulses when you push too hard. For the price of this one, I feel like it’d be nice if it had that feature too.- The glass charger is neat and a convenient storage place, but I think I’d prefer if it could stand straight up in its stand rather than let it lean against things.- The green charging indicator light is very dull and difficult to see in the daylight.

Great Clean, for Design Not so Much

The Philips HX939W feels great in the hand and provides a superior tooth-cleaning experience. One reason I bought it to replace my obsolescent Philips brush was the name – using Diamond Clean brushes had always given me a great clean, with smooth teeth like you get from a dentist. This new model is multi-functional, up to 3900 rpm, and comes also with an advanced brush which I use alternately with a Diamond Clean. You can charge the head by setting it in a provided glass, sort of nice. I don’t use the glass for other purposes. The system also comes with attachment so you can connect to your laptop to charge while traveling (I don’t need that) and a small travel bag (I don’t need that). What the system does not provide is something I wanted but had to purchase separately from Philips – a small container that holds two brushes and dries them with ultra-violet light, which is almost a requirement for storage in the bathroom. So, I would give five stars for end results, but three stars for product design, averaging out to a four. This should last for years.
