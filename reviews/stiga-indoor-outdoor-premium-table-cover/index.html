    Product Description

Durability The STIGA Indoor/Outdoor Premium Table cover is a great accessory to protect your table from wear and tear. This cover is made of the same durable material found in most high-quality automobile covers, offering exceptional protection for your table tennis table.

Secure Fit in Storage Position This table cover is 66.5”L by 61”W and is designed to fit most tables in their storage (vertical) position. With elastic hems, this table cover offers a snug fit to ensure it won’t slide off the table.

Indoor and Outdoor Use This table cover is designed for use indoors and outdoors. Use it indoors to prevent scratches, dust collection, and minor imperfections. Use it outdoors to keep leaves and dirt off your outdoor table and discourage rusting and discoloration from the sun.

Although this table cover can be used both indoors and outdoors, this cover should not be used to store an indoor table outside. Tables that are made for indoor use should always be stored in a climate-controlled space and should never be stored outside or in garages, covered deck or patios as moisture will cause the table top to warp. Even with a table cover warping may occur if an indoor table is stored outdoors.

About STIGA STIGA has been a world leader in the sport of table tennis for over 70 years. Originally renowned for innovative blade production, STIGA products set the standard for both table tennis accessories and tables. STIGA is continually working to develop innovative products by partnering with world class professional table tennis players. STIGA sponsorships include the Chinese National Team and the Swedish National Team which are the most respected and successful programs in the sport of table tennis.
