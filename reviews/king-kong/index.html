Beautifully done UHD with HDR

There are plenty of reviews around about the movie itself. If you like this movie then what is important is the quality and value of this particular release. So quality first as that will help decide its value. The UHD version of this movie has HDR. If you have a 4K display with HDR then you are going to get an extremely vivid and detailed experience, one that is significantly more rewarding than the regular Blu-Ray can deliver. This release is one of the most obvious examples of the superiority possible with UHD and HDR. The colors are more vivid with less bleeding and the contrast is just outstanding. As to the value of this release you must have the proper system for it. I should mention that the UHD disc contains both the original cinematic release and the extended version as well adding to the value of this set. As is typical of UHD releases it also contains a standard Blu-Ray disc and a digital copy as well. While I prefer the original 1933 version over all others, this is a pretty slick movie. I believe Kong looks better in this film than in the recent Kong: Skull Island. So if you like this movie and have the equipment it is a solid value.

Worth the upgrade to 4K

The 4K disc is worth the money. Gives you the option to watch the extended cut which was a pleasant experience. The added picture definition of 4K plus the punch brought by UHD made watching this movie a whole new experience. Ended up like this more than I thought I would. BTW I already owned the blue ray so there was some hesitation involved in making the decision to buy this. However, if you own a 4K TV and blue ray player its worth the upgrade. Cheers!
