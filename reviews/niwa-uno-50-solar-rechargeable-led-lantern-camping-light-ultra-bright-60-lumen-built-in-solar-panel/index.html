Lightweight, packable, versatile and BRIGHT!

I like several things about this product. It is light, packable, versatile, and surprisingly bright! We put it outside to charge when it arrived yesterday morning and by dinner, we were commenting on the brightness of the lamp and how nice it was to be able to angle the light. The base is very sturdy, or you can hang it up. The only drawback is that you can’t leave it outside–the instructions state it is only for indoor use. I knew this when I bought it, but it seems that with so many weatherproof solar lights available, this one might have been built to withstand weather as it’s for camping. That said, it is still a great light, whether around the house or in the tent. Not sure yet how long the life of the charge is, but so far, I’m impressed with the design and many uses!
