Product Reviews

WARNING: This drive gets HOT!!!

I’ve been impressed with the size and speed of the drive so far, and it works great as a keychain, but the drive itself gets VERY hot and is difficult to touch, especially closer to the port. I would recommend gripping the back of the drive near the plastic loop when removing the drive from your computer. Just thought I would warn everyone since I could see getting burned.

Small flash drive with good speed

Overall this is a great flash drive, it gets around 140MBs read and 40MBs write (Slightly slower than the 64GB version). It’s not quite as fast as the SanDisk extreme version but it’s the fastest metal drive I’ve found, you can see a benchmark in the attached image.The metal design seems sturdy and should hopefully hold up to being thrown around with my keys. My only complaint would be that the drive is part metal (silver portion) and the key ring part is plastic (black/grey portion in picture). I was hoping it would be all metal but the plastic key ring seems sturdy and it should hold up over time. One other note is that the drive can get very warm when doing large transfers, it’s to be expected for a small metal drive like this but it’s worth mentioning.

Fast and large enough to run VHDX’s for demo purposes on your laptop.

I run Hyper-V on my Windows 10 laptop that has an SSD with 16GB RAM and Core i7 processor. I keep my WIndows 2012R2 VHDX’s on the SanDisk USB 3.0 Flas Drive for SSD performance on my Hyper-V servers! It gets a little warm, within spec they say, but to keep it cool I stuck some heatsinks for a Rasp-Pi on it. Since the case is metal, heat transfer is excellent and I no longer worry about the temp of my external, thumb-sized SSD!

The Need For Speed!

This drive has been great thus far. Very fast and easily recognizable in both Windows 7 and 8. As a word of warning, file transfers are so fast, both up and down, that you may automatically think this drive is broken. On the contrary, it’s not. It really is that fast. The only true problem this drive has is it’s just 64GBs in size. So chances are the 128GB and higher capacities perform just as well as this drive. This drive is great for the price/performance ratio too. If 64GBs isn’t too small capacity for you then get this drive, you won’t regret it.

Very fast, great capacity

I bought four of these for a work project upgrading software on about 100 user desktop PCs; the only issue we had was actually that most of the PCs were USB 2.0 and our USB-device scan protocol took longer than we’d like, but that’s nothing against these devices, just the hardware. They were perfect for the job, because we couldn’t connect too many of the machines to the host server at once or it would get bogged down transferring the software. Four people and we still probably wouldn’t have finished in one day without these things. On the USB 3.0 machines, they were super fast!

Nice Security Software, but no USB cap and gets hot

Usable storage size is 115 GB, which is expected (the drive is listed as 128 GB, but some of the space is used for, I’m assuming, their security software). Speaking of security, the included software is pretty nice. I carry my drive to and from school/work and it’s reassuring to know that if I lose it, my files will remain private. As other users have mentioned it gets really hot even after just a few minutes of use. I wouldn’t say so hot you can’t touch it, but… it’s a bit alarming, even if it’s within the safety margins/standards SanDisk has cited. It cools down pretty quickly though. I haven’t noticed slow downs with prolonged use, but maybe I haven’t used it for as long as some other users. I got it as a bundle with another item on Prime Day, so I haven’t done much research on other flash drives of this size and I can’t really recommend it vs other drives out there. However, if you are on the fence about this one vs another drive, this isn’t a bad drive and the security software is neat. There isn’t a cap for the USB, so I’m a bit worried about dust getting in, but I plan on carrying it inside a case with other electronics. There’s also a loop on it to carry it on a lanyard if you want.Note about the security software: I kept getting a failure to run error when I tried to update the software, but a quick Google search found me a solution by a kind and clever user out there: “[I] renamed the executable [to] ‘SanDiskSecureAccessV3_Win.exe’ - relaunched it and it updated fine.“Pros:- Easy to use security software- Good storage size, 115 usable GBCons:- No USB cap- Gets hot

Slow File Writing Speed for Pictures and Video

I’m a photographer and videographer that has to use flash drives to transfer media to clients and every now and then, I like to try out new flash drives. I used this one and found that image and Video file transfers were a bit slow on this flash drive compared to a lot of other models. If you plan on transferring a lot of files, you may want to look elsewhere. The read capability of this flash drive is just fine but the file speed transfer of pictures and videos isn’t very appealing. However, if you’re just transferring basic word documents, music, or any other files like that, I don’t think you’ll have any issues or problems.

Really fast, but really hot.

Pros: 1) Very, very fast for the price. 2) Quite compact for a 64 GB drive.Cons: 1) It has no cap and no case, so you have to be careful to prevent damage to the unit. 2) I’d forgotten to check before purchasing as to whether it has an in-use indicator light; it does not. 3) It gets quite hot when transferring large amounts of data, which can’t be good for either the flash drive or the PC. When removing it, try to touch the plastic grip at the end, and not the metal body; and don’t put it down before allowing it a few seconds to cool down.
