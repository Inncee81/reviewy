This is a great little projector

This is a great little projector, quite portable and bulb replacement easy/cheap (we keep a few on hand). We’ve used this all over - great for PC display at matching resolution, but just recently found out that for movies (netflix/etc) better to send 1080p signal to it, ala Roku or similar - than streaming via PC - The projector seems to process (i.e. adapt) the 1080p to its native resolution really well - project it on a nice screen and it is pretty exceptional for a WXGA. Maybe a hidden feature for some, but as we use this mainly for portable presentation, we have to switch back and forth from high to low power modes quite a bit, which is somewhat menu intensive by default - there’s a feature to program the ‘User’ button on the remote, for us a great place to put this function - so now we can switch that as quickly as the color modes, which is really helpful in live venue. We have more hours logged in ECO than not, which says something for the brightness of the unit, as we choose this setting solely on visual effect when in the field. Final note, sound options are rather limited, but many devices such as an HDMI sound splitter can rectify this. We actually bought a second as a backup, though haven’t needed it yet. Which is a good thing. Overall, this projector sits high on my list of good purchases.
