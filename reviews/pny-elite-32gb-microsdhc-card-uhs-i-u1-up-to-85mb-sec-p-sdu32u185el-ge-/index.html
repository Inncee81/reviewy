Questionable origin, good sequential speeds, poor 4k speeds.

This review is for the 200gb PNY Elite MicroSD card, purchased this directly from Amazon (not a third party seller.) The first thing I do when purchasing any flash media these days is test integrity and check for fakes. It’s sad that we can’t purchase even from reputable dealers anymore without expecting a high chance of counterfeit media, but that’s our reality. I’ll include my test results below and let you judge this product for yourself.First test: Ran a Quick Size Test using fakeflashtest. Results were ok, so at least it seems to be the correct capacity. So far so good.Second test: Inspected card info using Chip Genius. No matching records found for the vendor or the part number, which is just slightly suspicious. If it were only the part number I wouldn’t be worried, since it’s possible that the 200gb version is just not in their database yet. It’s unlikely that PNY or its legitimate suppliers would not return a hit though, as they’re definitely not a new or unknown company.Third test: Formatted using a Galaxy S7 (the device intended to be used with it) and ran the app SD Insight. Origin unknown, Manufacturer unknown, OEM/Application ID is Barun Electronics. According to Google, Barun is a semiconductor manufacturer based in Korea. It’s possible that PNY sources their chips from them and doesn’t bother to change the manufacturer ID, but I wasn’t able to find any information that would confirm or deny this. The stamp on the packaging indicates it’s made in Korea, so at least that’s consistent.Fourth test: Popped it back into my computer (plugged directly into a Transcend USB 3.0 card reader in a USB 3.0 port) and ran CrystalDiskMark to test speeds. Sequential write was a solid 30.76 mb/s, but drops sharply after that.
