    Product Description

The Raspberry Pi 3 Model B is the third generation Raspberry Pi. This powerful credit-card sized single board computer can be used for many applications and supersedes the original Raspberry Pi Model B+ and Raspberry Pi 2 Model B.

Whilst maintaining the popular board format the Raspberry Pi 3 Model B brings you a more powerful processor, 10x faster than the first generation Raspberry Pi.

Additionally it adds wifi & Bluetooth connectivity making it the ideal solution for powerful connected designs.

Skip the bundle, find your own parts

To be fair, the Raspberry Pi itself seems to work fine, but several other pieces of this kit were busted. When first starting things up, I was running into mysterious errors. It turned out that the SD card that came with the bundle was corrupt. I was able to use an SD card I already had to get things running. The heat sinks seem to work, but the thermal grease doesn’t hold them in place, so they’re prone to slide around, which is a real problem for such a lightweight system. There are no instructions on how to plug in the LED case light, and everything I could find showed that having the red and black wires connected prevent plugging it in to the recommended pins. I was able to find a combo that lights up when the system is powering on, but not while it’s running. And after a week, the power supply failed and I had to figure out another alternative that could provide the recommended 2 - 2.5 A power.While I’m enjoying using the Pi itself, I’m disappointed with virtually every other aspect of this kit.

Novice computer guy here able to turn RPi into media center and retro gaming console! Love this thing!

I love my Raspberry Pi 3! If you are new to RPi and Linux like me, you may have some of a learning curve to deal with. But once I figured it out or found a detailed tutorial to help me, I was able to turn this into a Media Center running Kodi, a retro gaming console running Retropie and Emulation Station, as well as an open source desktop OS. It’s been fun and a little frustrating learning how to program all of this and build it, but it’s completely worth it now.This bundle came with everything I needed to get started (you may want to invest in a USB or wireless keyboard and mouse if you don’t have one). My only 2 gripes about this bundle were the MicroSD card it came with and the card reader. The card reader is super cheap and came completely apart when I was removing it from me computer’s USB slot at one point. The MicroSD stopped working after a few days of use. I contacted SJ Deals (the store that sold the bundle through Amazon) and they gave me a 20% refund to purchase a new card because they were out of 32GB microSD cards.Other than that, I have really enjoyed putting my RPi together and I enjoy using it. I love all of the possibilities this thing has to offer, too. I’m excited to see what additions people come up with.

This kit has everything you need to get started with …

This kit has everything you need to get started with a Raspberry Pi. The Raspberry Pi 3 is extremely fast and has WiFi built in along with an Ethernet port and 4 USB ports. This kit uses a thermal heatsink compound and 2 heatsinks. Other kits I’ve had use a peel and stick thermal tape to attach the heatsinks. This kit came with a preformatted 32 Gb Class 10 micro SD card with adapter. The micro SD card has NOOBS and Raspian already on the card so it is plug and play. The only thing missing is a keyboard and mouse. Pick up a mini wireless keyboard with touchpad and you’ve got everything you need. I plan on using this as a media center and will install Kodi on it.
