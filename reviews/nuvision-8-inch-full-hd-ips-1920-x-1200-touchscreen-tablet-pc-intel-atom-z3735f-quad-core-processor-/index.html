Product Reviews

actually a pretty neat tablet :)

This is the second of these tablets I’ve owned; I bricked the first one (my fault) and the one I’ve got now I’ve had for almost two months. My observations -* It performs as advertised. Performance is acceptable but this is not and never will be a gaming rig. I use mine for casual web surfing and watching the occasional video and for that it performs just fine.* Battery life is just okay. If you try to charge the device from a computer’s USB port you’ll find out in a hurry that the tablet draws more power than a USB port can provide. I use a 2.5 amp USB charger which works just fine.* The device has a single micro-USB port that can be used for data or charging but not both at the same time. I have an unpowered USB hub I attach to the device if I want to use a real mouse and keyboard.* I run Windows 10 on this one but have successfully installed Linux on one - not recommended. Driver support ain’t there with any mainstream Linux distribution and it takes a fair bit of messing around with the thing to get all the hardware working. I did get all the hardware working except the screen refused to rotate because the tablet refused to load a modesetting video driver.* Gorgeous screen. I have to run Windows at 175% magnification and some applications don’t respect that setting so some applications display really tiny text. I’ve had no problem with google chrome, firefox or any Microsoft application.* Is it worth the price? Yup - but it’s not a gaming rig and the battery isn’t gonna last six hours. I’ve never run the device flat but Windows says battery should last about 3.5 hours.
